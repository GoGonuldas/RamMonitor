# Play Store Copy Pack - RAM Monitor

## Kısa Açıklama (TR)
Anlık RAM, uygulama bazlı bellek ve canlı ağ kullanımını tek ekranda izleyin.

## Short Description (EN)
Monitor real-time RAM, per-app memory, and live network usage in one app.

---

## Uzun Açıklama (TR)
RAM Monitor, cihazınızın bellek kullanımını sade ve hızlı bir arayüzle izlemenizi sağlar.

Öne çıkan özellikler:
- Anlık toplam RAM kullanımı (kullanılan/boş RAM ve yüzde)
- Uygulama bazlı RAM listesi
- Geçmiş sekmesinde kullanım grafiği ve özet istatistikler
- Canlı ağ trafiği (indirme/yükleme hızı)
- Uygulama bazlı 24 saatlik ağ kullanımı
- Ön plan servis ile düzenli izleme ve bildirim desteği

İzinler neden gerekli?
- Kullanım Erişimi (PACKAGE_USAGE_STATS): Uygulama bazlı istatistikleri göstermek için
- Bildirim izni (POST_NOTIFICATIONS): İzleme bildirimlerini göstermek için
- Foreground Service izinleri: Arka planda güvenilir izleme sağlamak için
- Boot izni (RECEIVE_BOOT_COMPLETED): Cihaz yeniden başladıktan sonra izlemeyi sürdürebilmek için

Gizlilik:
- Veriler cihaz üzerinde işlenir.
- Kişisel veri toplama, satma veya üçüncü taraflarla paylaşma yapılmaz.

---

## Long Description (EN)
RAM Monitor helps you track device memory usage with a clean and fast interface.

Key features:
- Real-time total RAM usage (used/free memory and percentage)
- Per-app RAM usage list
- History chart with summary statistics
- Live network traffic (download/upload speed)
- Per-app 24-hour network usage
- Foreground service for continuous monitoring and notifications

Why permissions are needed:
- Usage Access (PACKAGE_USAGE_STATS): To show per-app usage statistics
- Notification permission (POST_NOTIFICATIONS): To display monitoring notifications
- Foreground service permissions: For reliable background monitoring
- Boot permission (RECEIVE_BOOT_COMPLETED): To resume monitoring after device restart

Privacy:
- Data is processed on-device.
- No personal data is sold or shared with third parties.

---

## İzin Beyanı (TR)
Bu uygulama, kullanıcıya cihaz RAM kullanımı ve uygulama bazlı ağ/RAM istatistiklerini göstermek için Kullanım Erişimi (PACKAGE_USAGE_STATS) iznini ister. Bu izin yalnızca uygulama içindeki analiz ekranlarında (Uygulamalar, Ağ ve Geçmiş sekmeleri) istatistik üretmek için kullanılır. Uygulama bu verileri üçüncü taraflarla paylaşmaz ve reklam/izleme amacıyla kullanmaz.

## Permission Declaration (EN)
This app requests Usage Access (PACKAGE_USAGE_STATS) to display device RAM usage and per-app RAM/network statistics. The permission is used only for in-app analytics screens (Apps, Network, and History tabs). The app does not share this data with third parties and does not use it for advertising or tracking.

---

## Data Safety Notes
- App collects personal data: No
- Data shared with third parties: No
- Data sold: No
- Data processed on-device for app functionality: Yes
