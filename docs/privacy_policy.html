Genel Bakış

RAM Monitor; cihaz belleği ve ağ kullanım istatistiklerini kullanıcıya göstermek amacıyla geliştirilmiş bir izleme uygulamasıdır. Uygulama herhangi bir kişisel veri toplamaz, işlemez veya dış sunuculara göndermez.

Toplanan Veriler

RAM Monitor aşağıdaki kişisel verileri toplamaz:

  • Ad, soyad, e-posta veya telefon numarası
  • Konum bilgisi (GPS veya ağ tabanlı)
  • Fotoğraf, kamera veya mikrofon verisi
  • Hesap veya kimlik bilgisi

Uygulama yalnızca aşağıdaki teknik verilere cihaz üzerinde erişir:

  • Cihaz RAM kullanım değerleri
  • Yüklü uygulamaların kullanım istatistikleri (Usage Stats API)
  • Uygulama bazlı ağ kullanım miktarları

Verilerin Kullanımı

Teknik veriler yalnızca şu amaçlarla kullanılır:

  • Uygulama ekranlarında istatistik ve grafik göstermek
  • Geçmiş kullanım grafiklerini oluşturmak
  • Yüksek RAM kullanımında yerel uyarı bildirimi göndermek

Veri Paylaşımı

RAM Monitor hiçbir veriyi üçüncü taraflarla paylaşmaz, satmaz veya reklam/izleme amacıyla kullanmaz.

Veri Depolama

Tüm veriler yalnızca cihaz üzerinde yerel olarak saklanır. Uygulama herhangi bir sunucuya veri göndermez.

Çocukların Gizliliği

RAM Monitor, 13 yaşından küçük çocuklara yönelik değildir. Uygulama hiçbir kullanıcıdan kişisel veri toplamaz.

🇬🇧 English

Overview

RAM Monitor is a monitoring application designed to display device memory and network usage statistics to the user. The app does not collect, process, or transmit any personal data to external servers.

Data Collection

RAM Monitor does not collect:

  • Name, email address, or phone number
  • Location data (GPS or network-based)
  • Photos, camera, or microphone data
  • Account or identity information

The app accesses only the following technical data on-device:

  • Device RAM usage values
  • Installed app usage statistics (Usage Stats API)
  • Per-app network usage totals

How Data Is Used

Technical data is used only to:

  • Display statistics and charts in the app
  • Generate usage history graphs
  • Send local alert notifications for high RAM usage

Data Sharing

RAM Monitor does not sell or share any data with third parties for advertising or tracking purposes.

Data Storage

All data is stored locally on the device only. The app does not send data to any remote server.

Permissions Used

PermissionPurpose
PACKAGE_USAGE_STATSTo show per-app usage statistics
FOREGROUND_SERVICEFor continuous background monitoring
FOREGROUND_SERVICE_DATA_SYNCTo periodically update RAM data
POST_NOTIFICATIONSTo display monitoring notifications
RECEIVE_BOOT_COMPLETEDTo resume monitoring after device restart
INTERNETGeneral Android network compatibility (no data sent)

Children's Privacy

RAM Monitor is not directed at children under 13 years of age. The app does not collect personal data from any user.
